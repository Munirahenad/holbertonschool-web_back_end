# ES6 Promise

This project focuses on learning and practicing JavaScript ES6 Promises, async handling, error management, and working with multiple promises.

## Learning Objectives

By completing this project, I learned how to:

- Create and use JavaScript Promises
- Handle resolved and rejected promises
- Use `.then()`, `.catch()`, and `.finally()`
- Work with `Promise.all()`
- Work with `Promise.race()`
- Handle multiple settled promises with `Promise.allSettled()`
- Throw and handle errors using `try/catch`
- Build asynchronous JavaScript functions

## Requirements

- Ubuntu 20.04 LTS
- Node.js
- npm
- ES6 syntax
- All files should end with a new line

## Files

| File | Description |
|---|---|
| `0-promise.js` | Returns a Promise object |
| `1-promise.js` | Returns resolved or rejected Promise |
| `2-then.js` | Handles API response using then/catch/finally |
| `3-all.js` | Handles multiple successful promises |
| `4-user-promise.js` | Returns a resolved user object |
| `5-photo-reject.js` | Returns a rejected Promise |
| `6-final-user.js` | Handles multiple settled promises |
| `7-load_balancer.js` | Returns the first resolved Promise |
| `8-try.js` | Throws an error when dividing by zero |
| `9-try.js` | Uses try/catch/finally for error handling |
