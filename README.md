# Holberton School Web Back-End

## Overview

This repository contains my projects for the **Holberton School Web Back-End** track.  
It includes a collection of Python projects that build core back-end programming skills, with a focus on:

- Python fundamentals
- Type annotations
- Asynchronous programming
- Async generators and comprehensions
- Python object behavior and memory model

These projects are designed to strengthen problem-solving, code readability, and understanding of how Python works internally.

---

## Objectives

By completing these projects, I am improving my ability to:

- Understand Python object behavior
- Differentiate between mutable and immutable objects
- Use type annotations correctly
- Write clean and maintainable Python code
- Work with asynchronous programming using `asyncio`
- Execute multiple coroutines concurrently
- Create and manage tasks
- Use asynchronous generators
- Write async comprehensions
- Document code clearly and professionally

## Topics Covered

### Python Variable Annotations
This section focuses on learning how to use **type hints** in Python for:

- Function parameters
- Return values
- Variables
- Collections such as `List`, `Tuple`, `Union`, `Callable`, `Iterable`, and `Sequence`

### Python Async Function
This section introduces the basics of **asynchronous programming** using `asyncio`, including:

- Writing coroutines with `async def`
- Using `await`
- Running concurrent coroutines
- Creating tasks with `asyncio.Task`
- Measuring execution time for asynchronous code

### Python Async Comprehension
This section focuses on:

- Asynchronous generators
- Async comprehensions
- Running multiple asynchronous operations in parallel with `asyncio.gather`

### Python - Everything is object
This section explores Python‚Äôs object model and internal behavior, including:

- Objects and references
- Assignment and aliasing
- Identity and equality
- Mutable and immutable types
- How Python passes variables to functions
