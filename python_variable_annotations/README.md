# Python Variable Annotations

This project is part of the **Holberton School Web Back-End** curriculum.  
It focuses on learning **type annotations in Python**, including function annotations, variable annotations, and the use of types from the `typing` module.

## Learning Objectives

At the end of this project, I should be able to explain:

- What type annotations are in Python
- How to annotate function parameters and return values
- How to annotate variables
- How to use types such as `List`, `Tuple`, `Union`, `Callable`, `Iterable`, and `Sequence`
- How type hints improve code readability and maintainability

## Files

### `0-add.py`
Contains a function `add` that takes two floats and returns their sum as a float.

### `1-concat.py`
Contains a function `concat` that concatenates two strings and returns the result.

### `2-floor.py`
Contains a function `floor` that returns the floor of a float as an integer.

### `3-to_str.py`
Contains a function `to_str` that converts a float to its string representation.

### `4-define_variables.py`
Defines annotated variables:
- `a` as an integer
- `pi` as a float
- `i_understand_annotations` as a boolean
- `school` as a string

### `5-sum_list.py`
Contains a function `sum_list` that takes a list of floats and returns their sum.

### `6-sum_mixed_list.py`
Contains a function `sum_mixed_list` that takes a list of integers and floats and returns their sum as a float.

### `7-to_kv.py`
Contains a function `to_kv` that returns a tuple containing a string and the square of a number as a float.

### `8-make_multiplier.py`
Contains a function `make_multiplier` that returns a function used to multiply a float by a given multiplier.

### `9-element_length.py`
Contains a function `element_length` that returns a list of tuples containing sequence elements and their lengths.
