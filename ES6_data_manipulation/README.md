# ES6 Data Manipulation

This project focuses on ES6 data manipulation using JavaScript modern features such as arrays, maps, sets, weak maps, typed arrays, and array methods.

## Learning Objectives

By the end of this project, I learned how to:

- Use `map`, `filter`, and `reduce`
- Work with typed arrays
- Create and manipulate `Set`
- Create and update `Map`
- Use `WeakMap`
- Export and import functions using ES6 modules

## Requirements
- Ubuntu 20.04 LTS
- Node.js
- npm
- Files must use `.js` extension

## Files

| File | Description |
|---|---|
| `0-get_list_students.js` | Returns an array of student objects |
| `1-get_list_student_ids.js` | Returns an array of student IDs |
| `2-get_students_by_loc.js` | Filters students by location |
| `3-get_ids_sum.js` | Returns the sum of student IDs |
| `4-update_grade_by_city.js` | Updates student grades by city |
| `5-typed_arrays.js` | Creates an ArrayBuffer with an Int8 value |
| `6-set.js` | Creates a Set from an array |
| `7-has_array_values.js` | Checks if all array values exist in a Set |
| `8-clean_set.js` | Cleans and joins Set values based on a starting string |
| `9-groceries_list.js` | Returns a Map of grocery items |
| `10-update_uniq_items.js` | Updates unique grocery quantities |
| `100-weak.js` | Tracks API calls using WeakMap |
